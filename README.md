# ProjetoFinalPHP

Faremos um projeto de controle de estoque e de vendas de uma loja "X" de móveis. o cliente terá acesso a imagens do produto e de uma descrição com preço, material, etc.
a ideia principal é fazer com que assim que o "cliente" efetue a compra no site, automaticamente dê baixa no estoque no banco de dados, para que o "dono" tenha um controle refinado do que entra e sai do estoque da loja.
